MSMBuilder
==========

MSMBuilder is a tools for constructing Markov state models of biomolecular conformational dynamics.

MSMBuilder has been designed to provide both ease of use and
versatility. To facilitate the workflows of both novices and experts, we
have designed MSMBuilder with two modes of operations:

#. MSMBuilder is a set of python scripts.

#. MSMBuilder is a library.

Python scripts allow most users to work without writing a single line of
Python code. Advanced users can write their own Python scripts using the
MSMBuilder library.
